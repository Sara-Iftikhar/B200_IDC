# Prerequisites

Basic knowledge of macOS, molecular biology, genetics, and microbiology.

Before Day 1, please install:

- **[R](https://cran.rstudio.com/)** and **[RStudio](https://posit.co/download/rstudio-desktop/)**
- **[Miniconda](https://www.anaconda.com/download/success)**
- An **SSH client** for connecting to KAUST's HPC (IBEX) — built into the Terminal on macOS/Linux, or [PuTTY](https://www.putty.org/) on Windows

See the [Resources](resources.md) page for links to all other software used in the practicals.
